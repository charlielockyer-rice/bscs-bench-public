# Provided Baseball Statistics

There are four types of statistics included for each team.
1. The number of games a team won in a given season.
2. Aggregate player stats of all players on the team that season.
3. Aggregate player stats of all players leaving the team before the next season.
4. Aggregate player stats of all players joining the team before the next season.

For each of the statistics we have included below its column number in the statistics matrix, a short identifier for the statistic, and an english description of the statistic.
### **1. Number of Games won**

| Column # | Identifier | Description |
| --- | --- | --- |
| 0 | WON | Games won this year |
