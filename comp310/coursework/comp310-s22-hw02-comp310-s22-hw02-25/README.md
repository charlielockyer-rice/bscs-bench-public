## Application Notes:
To run, right click the BallController.java file, hover 'Run As', and select "Java Application".
The following is the exact text to be put in the text field to generate a ball, as well as a
description of the ball:
hw02.ball.DVDLogoBall: A ball which changes color when it hits a boundary, like the logo in the DVD idle screen
hw02.ball.BounceUpBall: A ball that forces perspective, appearing as if it is bouncing out of the screen when moving
right and as if it is bouncing away from the screen when moving left.
hw02.ball.SineSpeedBall: A ball whose velocity which oscillates according to randomly generated sine waves
