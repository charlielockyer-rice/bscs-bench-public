## Application Notes:
How this works is you type one of the following strategies in the text box:
1. RandomWalk
2. Color (as in demo)
3. Expanding (as in demo)
4. Gravity
5. Reverse
6. Straight (as in demo)
7. Teleporting
8. Exploding

You then add to list and can select it via the drop down. You can then make selected ball. To combine balls make sure you have one item in the top dropdown and one item in the buttom drop down and then click combine traits. Select the top drop down and select the new combined trait and make the ball. To make a swicher (defaulted to straight ball) just click make switcher button. To switch the behavior, put the intended behavior in the second drop down (i.e. buttom most) and click switch to watch ALL switcher balls switch to this new behavior. Then click clear balls when ever you want all the balls cleared from the display.
